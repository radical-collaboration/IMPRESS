# Agentic workflow

* pip_requirements.txt - freeze output from a uv development env
* ProteinBindingPipelineAgent.py - state definition for the pipeline agent class (will not be used, but is a more familiar expression of the data requirements)
* prototyping.ipynb - notebook with agent prototypes in langchain
* run_agent.slurm - batch script with server init& run main.py
* vllm_server.sh - server script for running a localhost endpoint for tool users
